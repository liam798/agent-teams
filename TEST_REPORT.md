### ⚠️ 5. 已知问题

1. **WebSocket测试**: 需要安装 `websocat` 工具进行完整测试
2. **构建警告**: Three.js库较大（>500KB），建议代码分割优化
3. **依赖冲突**: 使用 `--legacy-peer-deps` 安装UI依赖

### 📝 6. 测试建议

#### 手动测试步骤：

1. **启动服务器**:
   ```bash
   npm run build
   node dist/cli.js ui
   ```

2. **访问主页**:
   - 打开浏览器访问 http://localhost:3000
   - 应该看到团队列表页面

3. **测试3D模式**:
   - 点击右上角的"普通模式"按钮切换到"3D模式"
   - 进入团队详情页面
   - 应该看到3D场景中的团队成员可视化
   - 使用鼠标拖拽旋转场景
   - 使用滚轮缩放

4. **测试创建团队**:
   - 点击"新建团队"按钮
   - 填写团队名称
   - 添加多个成员
   - 选择不同平台
   - 提交创建

5. **测试任务管理**:
   - 进入团队详情页
   - 查看任务看板
   - 测试任务状态切换
