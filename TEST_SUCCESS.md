## 🔍 调试信息

如果3D模式仍然不工作，请：

1. **打开浏览器控制台** (F12)
2. **查看日志输出**，应该看到：
   - `[Members3D] 组件渲染: { is3D: true, membersCount: 5 }`
   - `[Scene] 场景初始化: { membersCount: 5 }`
   - `[Scene] 计算位置完成: { count: 5 }`
   - `[Members3D] Canvas created successfully: { vendor: ..., renderer: ..., version: ... }`
   - `[MemberAvatar] 成员名 几何体创建成功`

3. **检查WebGL支持**：
   - Chrome: 访问 `chrome://gpu`
   - Firefox: 访问 `about:support`
   - 确保WebGL已启用

4. **检查错误信息**：
   - 如果有错误，ErrorBoundary会显示错误信息
   - 控制台会输出详细的错误堆栈
